# Task Manager (Core Java)

## 📌 Project Overview

Task Manager is a console-based Java application that helps users manage their daily tasks. Users can add, edit, delete, search, and mark tasks as completed. The application also saves tasks to a file so that they are available the next time the program is opened.

---

## ✨ Features

* Add a new task
* View all tasks
* Edit an existing task
* Delete a task
* Mark a task as completed
* Search tasks by title
* Show pending tasks
* Show completed tasks
* Set task priority (HIGH, MEDIUM, LOW)
* Save tasks to a file
* Automatically load saved tasks when the program starts

## ▶️ How to Run

1. Open the project folder in VS Code.
2. Open the terminal.
3. Compile the project:

```
javac *.java
```

4. Run the application:

```
java Main
```
